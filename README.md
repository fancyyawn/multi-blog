# multi-blog

> 多人博客

## 功能
用户，博客，评论，图片

## 目标
作为各类技术栈学习和经验积累的平台。

## 技术
* koa、express
* SpringBoot、springMvc、springSecurity、SpringCloud(微服务)、springData
* ejs、handlebars、thymeleaf、semanticUI
* mysql、mongodb等技术。
